## 📌 Project Overview

AI TaskGenie is an AI-powered productivity companion designed to help students, professionals, and entrepreneurs stay organized and productive.

Instead of acting as a simple reminder app, AI TaskGenie intelligently prioritizes tasks, generates personalized AI roadmaps, tracks productivity, provides analytics, and helps users complete their goals before deadlines.

This project was built for the **Coding Ninjas × Google for Developers – Vibe2Ship Hackathon** under the problem statement **"The Last-Minute Life Saver"**.
